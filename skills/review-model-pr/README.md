# Review Model PR Skill

Use this skill in repositories that contain [MetaEd models](https://github.com/Ed-Fi-Alliance/MetaEd) (e.g., Ed-Fi-Model) when asked to review pull requests or staged changes related to the model. This skill provides a structured checklist to evaluate the quality and correctness of MetaEd changes, covering build validity, documentation, naming conventions, and more. It helps ensure that all contributions meet the standards required for successful integration into the codebase.
